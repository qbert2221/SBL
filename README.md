# SBL Mortgage - The Corporate Dividend Mortgage™

A modern, asset-powered approach to home financing for investors who want to buy real estate without liquidating their portfolio.

## Features

- Interactive 3D playbook showcase
- Real-time mortgage calculator
- Animated statistics and counters
- Comprehensive implementation guide
- Free educational resources

## Technology Stack

- Pure HTML/CSS/JavaScript
- No build process required
- Responsive design
- Optimized for performance

## Deployment

Deployed on Railway with automatic deployments from GitHub.

## Local Development

```bash
python -m http.server 8000
```

Then visit `http://localhost:8000`
